톰캣이란 ?
웹 애플리케이션 서버 .
자바 서블릿을 실행시키고 JSP 코드가 포함되어 있는 동적 웹페이지를 구동시켜주는 프로그램 .

Java 코드를 컴파일하여 HTML 문서로 덮어씌운 후 아파치에 전달하는 역할

Http: 운영체제가 가지고있는 Socket을 이용해 만들었다 시스템콜을 이용해 만들음 .

URL : 자원 접근 ,요청하는 주소
URI : 식별자 접근 
L은 Location (위치) 를 의미한다 .
웹서버 :  HTTP 또는 HTTPS를 통해 웹 브라우저에서 요청하는 HTML 문서나 오브젝트(이미지 파일 등)을 전송해주는 서비스 프로그램을 말한다
아파치 요청한 파일을 응답해주는 서버
톰캣은 아파치가 요청받은 파일 중 JSP 파일을 HTML로 컴파일하여 돌려주는역할을 한다
